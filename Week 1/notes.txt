Git hub commands

git init
git add --all
git commit -m "add comment here"
git push 

or if first update

git add --all
git commit -m "add comment here"
git remote add origin urlofrepo
git push -u origin master



git status -  to see which files git recognizes (need to add "git add" command to update any new changes) 

pull request - can review before pushing the changes onto the main branch
